Esri Server to Tableau

GeoJSON scrape tool

Use this tool to download GeoJSON files from public facing services to use with Tableau. Copy the rest service url from the item description page and paste above without a slash "/" at the end of the address. The tool will check for existing layers, and create a checkbox for each existing layer after "Check for Layers" is clicked. Then select what layers you would like to download using the checkboxes and click "Run".


Ex. https://services9.arcgis.com/RHVPKKiFTONKtxq3/arcgis/rest/services/USA_Wildfires_v1/FeatureServer
